# estatistica-descritiva-supermercado
Projeto desenvolvido durante o curso de Ciência de Dados da EBAC, aplicando estatística descritiva, manipulação de dados e visualização de informações com Python.

Descrição

Projeto desenvolvido como atividade prática do curso de Ciência de Dados da EBAC. O objetivo foi aplicar conceitos de estatística descritiva, manipulação de dados e visualização utilizando Python.

A análise foi realizada sobre uma base de produtos de supermercado, explorando preços, descontos, categorias e marcas para identificar padrões e gerar insights.
